ABOUT
HEY! THAT'S MY FISH! is a 2-4 player game of strategy. Each player has 2-4 penguins and the goal is to catch as many fish as possible.

HOW TO PLAY
1. Create a new game room by entering your name and get an invite code to invite players.

2. If you have an invite code, join the room using the same.

3. Once a room has 2-4 players, you can start the game.
