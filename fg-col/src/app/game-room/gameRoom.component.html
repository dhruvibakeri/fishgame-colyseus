ABOUT
HEY! THAT'S MY FISH! is a 2-4 player game of strategy. Each player has 2-4 penguins and the goal is to catch as many fish as possible.

HOW TO PLAY
1. Create a new game room by entering your name and get an invite code to invite players.

2. If you have an invite code, join the room using the same.

3. Once a room has 2-4 players, you can start the game.

4. Once the game has started, each player places all their penguins on tiles according to their turn.

5. After all penguins have been placed, players can start moving their penguins to their desired tile (one that would get them the most fish).

6. A penguin can only move in straight lines (across the edges of the hexagon tile).

7. After each move, the number of fish on the tile the penguin moved from is added to the player's score.

8. The game ends when none of the players have any moves left (the player with the highest score wins).
